# PrimeVue Verona

PrimeVue Verona theme template for Vue.js applications.

## Overview

This repository contains the Verona application template (v5.1.0) for PrimeVue, a popular UI component library for Vue.js. Verona provides a professional admin dashboard layout with sidebar navigation and responsive design.

## Contents

- `verona-vue-5.1.0.zip` - Complete Verona template package for Vue.js

## Getting Started

1. Extract `verona-vue-5.1.0.zip`
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
4. Build for production: `npm run build`
